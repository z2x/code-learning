字体系列

  • Serif字体：
    这些字体成比例，而且有上下短线。如果这些字体中的所有字符根据其不同大小有不同的宽度，则称这些字体成比例。
    如：Times，Georgia，New Century Schoolbool。
  • Sans-serif字体：
    这些字体成比例，但没有上下短线。
    如：Helvetica，Geneva，Verdana，Arial，Univers。
  • Monospace字体：
    这些字体不成比例，通常用于模拟打字机打出的文本、老式点阵打印机的输出，或者老式视频显示终端的输出。每个字符的宽度完全相同。可能有上下短线，也可能没有。
    如：Courier，Courier New，Andale Mono。
  • Cursive字体：
    模仿人的手写体，主要由曲线和serif字体中没有的笔画装饰组成。
    如：Zapy Chancery，Author，Comic Sans。
  • Fantasy字体：
    除去以上几种字体特征的字体。
    如：Woodblock，Western，Klingon。

注意：
当字体名中有一个或者多个空格、或者字体名中包括#、%之类的符号，要在font-family声明中添加引号。且引号需要属性本身未曾使用过。

字体加粗

可以使用font-weight进行定义，其属性值主要为：

  • normal：默认值。定义标准的字符。
  • bold：定义粗体字符。
  • bolder：定义更粗的字符。
  • lighter：定义更细的字符。
  • 100~900：定义由细到粗的字符。400 等同于 normal，而 700 等同于 bold。
  • inherit：规定应该从父元素继承字体的粗细。

字体大小

可以使用font-size进行定义，其属性值主要为：

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large

字体风格

可以使用font-style进行定义，其属性值主要为：

  • italic(斜体):italic text sample.
  • oblique(倾斜):oblique text sample.
  • normal
  • inherit
