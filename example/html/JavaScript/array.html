数组

要创建新数组，需要声明一个变量名，并使用new关键字和array()函数来说明这是一个数组。

例如：

var myArray = new Array();

在数组中存储数据时，只需在定义数组时，把用逗号分隔的数据放在Array()后面的圆括号中。

例如：
var myArray = new Array("Paul",345,"Bob",99);

定义了数组之后，就可以在数组中保存数据了，每次使用新索引来保存数据时，JavaScript都会自动创建一个新的存储空间。
注意:第一个元素的索引为myArray[0].

多维数组

JavaScript只支持一维数组，不支持不支持多位数组。但是JavaScript允许在一个数组内部创建另一个数组，从诶模拟出多位数组。
