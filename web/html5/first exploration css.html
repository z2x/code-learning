元素内嵌样式 and 文档内嵌样式

元素内嵌样式 > 文档内嵌样式 > 外部样式.

如果同一个样式属性出现在具体程度相当的几条样式中，那么浏览器会根据其位置的先后选择所用的值。规则是后来者居上。在元素内嵌样式中使用如class="class1,class2",同时在文档内嵌样式中对.class1和.class2进行值定义，则浏览器对.class2进行显示。
在文档内嵌样式中的.class1和.class2的定义进行颠倒，则浏览器对.class1进行显示。

如果浏览器在直接相关的样式中找不到某个属性的值，就会求助于继承机制，使用父元素的样式属性值。在样式中使用inherit进行强行继承。

css中的颜色

  1. 直接使用规定的颜色名称；
  2. 十六进制：如black为#000000；
  3. 十进制：如black为0,0,0.
  4. 用RGB模型表示颜色：如：color:rgb(112,128,144);
  5. 用RGB模型表示颜色，外加一个用于表示透明度的a值，0表示全透明，1表示完全部透明;
  6. 用HSL模型表示颜色(hue:色相，saturation：饱和度，lightness：明度）：如：color:hsl(120,100%,20%);
  7. 用HSL模型表示颜色，外加一个用于表示透明度的a值，0表示全透明，1表示完全部透明;

css中的长度

绝对长度

  1. in：英寸
  2. cm：厘米
  3. mm：毫米
  4. pt：磅
  5. pc：pica（1pc=12pt）

相对长度

    本文档在文档内嵌样式中将正文样式设置为0.2in

  1. em：与元素字号挂钩

    2. 修改字体大小font-size:0.4in与所设置font-size:2em相符：2em=0.4in

  2. ex：与元素的x高度挂钩

    3. 这个单位指的是当前字体的x高度，也即字体基线到中线之间的 距离，一般与字母x的高度相当，通常1ex大致等于0.5em。

  3. rem：与根元素的字号挂钩

    4. 修改字体大小font-size:0.4in与所设置font-size:2rem相符：2rem=0.4in

  4. px：css像素
  5. %：另一属性的值的百分比
  6. gd：与网格（grid）挂钩
  7. vw：与视口（viewport）宽度挂钩，1vm=文档显示区域宽度的1%
  8. vmin：1vmin=最短视口轴长的1%
  9. vmax：1vmax=最大视口轴长的1%
  10. ch：与用当前字体显示的字符的平均宽度挂钩

CSS中的角度

  1. deg：度（0deg~360deg)
  2. grad：百分度(0grad~400grad)
  3. rad：弧度(0rad~6.28rad)
  4. turn：圆周(1turn=360deg)

css中的时间

  1. s：秒
  2. ms：毫秒（1s=1000ms）
